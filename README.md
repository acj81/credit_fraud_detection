# credit_fraud_detection

Machine Learning task - uses updated 2023 credit fraud dataset from Kaggle:
https://www.kaggle.com/datasets/nelgiriyewithana/credit-card-fraud-detection-dataset-2023

Dataset already cleaned, nothing to do there

Compares different model architectures - uses models from scikit-learn as appropriate level of simplicity and (crucially) interpretability (i.e. logistic regression).
